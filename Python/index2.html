Managing a project is a key aspect in any start up organization for a success run. Every Organization should have a criteria of how they need to handle each and every phase of a project. Below are phases and processes for a successful project in an NGO.

Project Inception

This is a process where one identifies a need and thinks through about it before starting any actions towards it

Project Planning

This is a phase where one gets to plan on the resources he/she would need to succesfully do a project This involvoles various means and process that one uses during the process.
